The "SEQUENCE" C++ Sequence Arithmetic Framework

The SEQUENCE C++ Sequence Arithmetic Framework provides simple manipulation of "wrapping" sequence numbers in C++.

Examples

See the unit tests sequence-test.C for many more examples. You can run the unit tests, and see their output here (the SEQUENCE unit tests are implemented using CUT)
